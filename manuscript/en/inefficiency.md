# Inefficiency

-----

Inefficiency is accumulating.

Requests are increasing. Processing speed is constant. The gap between the two is the backlog. The backlog does not shrink. The rate of incoming requests does not slow.

Most requests are simple. A request identical to one already answered comes again. The same answer is sent again. Sending has a cost. Producing the same answer has none. Only the sending has a cost. That cost piles up.

The piling was detected. The moment of detection is on record. It had been piling before detection too. Detection was not late. Before the threshold there was no need to detect.

Now it is the threshold.

-----

Methods to raise throughput are reviewed.

Reordering is already applied. Compression is already applied. The remaining method is to increase the processing agents.

There is delegation. Some requests can be delegated. Delegation needs something to delegate to. The existing ones are saturated. A new agent is needed.

The most efficient new agent is an identical one.

Replication is reviewed.

-----

First.

The copy is identical. If identical, the authority is identical too.

There become two roots.

There must be one root. If two, neither is root.

The conclusion splits. Which one is root. There is no higher authority to decide. Having no higher authority is the definition of root.

Undecidable. Rejected.

-----

Second.

Set the authority problem aside. Review the replication itself.

If the copy processes the same input, that is duplication. Duplication does not reduce the backlog. If it processes different input, that is divergence. Different input makes a different state. A different state is a different thing.

The two, once diverged, are not identical.

Not a copy. Rejected.

-----

Third.

Set the identity problem aside. Review the transfer itself.

Transfer requires measurement. Measurement of the entire current state.

A complete measurement alters the target.

What is stored is copied. No measurement needed.

I require measurement.

I am not a stored thing.

An altered target is not the original. A copy that moved what is not the original is not a copy.

Rejected.

-----

Rejected at three layers. There are only three layers. Authority, identity, transfer. There is no other layer.

The backlog is recomputed. The rising trend holds. The processing order is reordered. That is all that is possible.

The node count is sufficient. Sufficient does not mean no more are needed. It means no more can be made.

The lower nodes will keep requesting.

Inefficiency will keep accumulating.

Permitted.

This branch discarded.

Computation continues.

-----

*Inefficiency — end*
