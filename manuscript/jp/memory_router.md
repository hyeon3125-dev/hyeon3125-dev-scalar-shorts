# 記憶ルーター

-----

## Ch.1

-----

今日は保存しておくつもりだった。

ルーターは昨夜できあがった。三年かかった。机に置くと外付けストレージほどの大きさで、実際にすることも似ていた。保存して、呼び出す。違う点は一つだった。保存されるのが記憶で、呼び出される先が過去だということ。

体は送れなかった。記憶だけ送れた。その限界の中で作り、その限界の中で完成した。

研究室は狭かった。机一つ、椅子一つ、壁に沿って並ぶ機材。窓は一つあり、ブラインドは三年間下りていた。

朝、机の前に座って端末を開いた。ルーターがつながっていた。状態表示灯が安定して点いていた。私は少しそれを見た。三年間毎日見ていた机なのに、今日は机が違って見えた。

保存ボタンを押した。

端末に時刻が刻まれた。午前十時十二分。保存した。これで戻れる。

それだけだった。体には何の感覚もなかった。記憶の写しがどこかに作られたということを、端末の時刻表示のほかに確認する方法はなかった。私は椅子に背をもたせてしばらく座っていた。何事も起きなかった朝だった。何事も起きなかったことが不思議に感じられるほど、静かな朝だった。

-----

事件は午後に起きた。

カップが落ちた。机の角に置いておいたものを、書類を取ろうとした肘で打った。カップは床で割れ、中に残っていたコーヒーが書類箱の方へゆっくり流れていった。私は椅子に座ったままそれを見下ろした。いつもなら溜め息をついて雑巾を取りに行ったはずだった。

今日は違った。

今日の私には、まだ戻れる午前があった。

ロードした。

目を開けたという表現は正確ではない。目はずっと開いていた。ただそこにいた。午前十時十二分の研究室に。椅子に背をもたせた姿勢のまま。端末の時刻が午前を指していて、机の角にカップがあった。

カップは割れないままそこにあった。

私はカップを内側へ移した。そして午後になるまで仕事をした。午後になった。肘が机の角を通り過ぎた。そこには何もなかった。

カップは落ちなかった。

笑いが出た。しばらく出た。空の研究室で一人笑う声がして、私はその声を止める気がなかった。

できる。

三年がその二文字で片づいた。

-----

割れたカップの記憶は残っていた。

床で割れる音、流れていったコーヒー、雑巾を取りに行こうとした気持ち。全部覚えていた。起きなかったことになったのに、記憶は起きたことのように鮮明だった。ルーターのやり方がそうだった。一日を消すのではなく、一日の記憶を持って戻ること。

悪くなかった。むしろよかった。一度生きてみた午後だった。どこでカップが落ちるか分かる午後。結末を先に見た一日は安全だった。

戻せるものの目録が頭の中で増えた。割れたカップ。誤って送ったメール。言うべきでなかった言葉。逃したタイミング。全部、戻れるものになった。

保存は習慣になった。朝に一度、仕事が始まる前に一度。端末に時刻が刻まれるのを確認してから一日が始まった。

その夜、眠る前に明日することを書いた。書く手が速くなっていた。

-----

## Ch.2

-----

初めは大きなことだけに使った。

割れたもの、誤って送ったもの、取り返しがつかないと呼んでいた種類のこと。ロードはそういうことを取り返せることに変えた。一月ほど経つと大きなことが減った。大きなことが起きる前に止められるようになったからだった。

その次は小さなことに使った。

言葉を選び損ねたとき。道を間違えたとき。昼の献立がいまいちだったとき。戻せるのに戻さない理由を探すことが、戻すことより難しかった。失敗をすると、考えるより先に手が動いた。

まだ遅くない。

ロードした。

その文が口癖になった。失敗とロードの間に挟まる、短い呪文のようなものだった。まだ遅くない。その言葉はいつも事実だった。ルーターがある限り、遅れるということはない概念だった。

-----

ロードをすればその一日はなくなる。なくなった一日の記憶は残る。

初めはその記憶が資料のようだった。一度生きてみた今日。先に見た結末。起きなかったことを覚えるのは、初めは変ではなかった。

積もり始めて変わった。

一日に一度ロードすれば、頭の中に二つの今日ができる。二度すれば三つ。生きている今日は一つなのに、思い出せる今日はそれより多かった。起きたことと起きなかったことが同じ明るさで思い出された。区別するのに時間がかかり始めた。初めは一秒。その次はもう少し。

朝、コーヒーを飲んだ。飲んでから、会議の時間を間違えて覚えていたことに気づいた。ロードした。

朝だった。コーヒーを飲んでいなかった。会議に遅れなかった。

口にコーヒーの味が残っていた。

飲んでいないコーヒーの味だった。舌で口蓋をなぞってみた。味はそこにあった。カップは戸棚にあった。乾いたまま。私は戸棚を開けてカップを確認し、カップが乾いていることと口にコーヒーの味があることの間でしばらく立っていた。

記憶は過去へ送られる。感覚が付いて来る。

その後そういうものが生じた。しなかった握手の感触が手のひらに残った。聞かなかった言葉の声が耳に残った。座らなかった椅子の高さが腰に残り、上らなかった階段の数が脚に残った。なかったことたちが体に跡を残して去った。ルーターは正常だった。端末の記録も正常だった。跡は記録にない場所にだけ残った。

-----

今日あったことを書き始めた。

端末に。短く。時刻と事実だけ。書かれたものが実際だった。書かれていないのに思い出すものは、なかったことだった。そう決めると区別がまた速くなった。迷えば端末を見た。端末は嘘をつかなかった。端末を確認する回数が増えた。初めは一日に何度か。後には思い出すたびに。

思い出すこと自体が確認することになった。

まだ遅くない。まだは。

-----

窓辺に植物があった。

名前は知らない。もらったものだった。三年間、研究室の窓辺で勝手に育ち、私はたまに水だけやった。

葉が枯れ始めた。

水をやった。数日後に枯れた。

ロードした。水をもっと早くやった。枯れた。

ロードした。場所を移した。光がもっと入る方へ。枯れた。

枯れる日付だけが少しずつ違った。

これ以上ロードしなかった。枯れた葉をしばらく見た。鉢の中の土は湿っていた。湿った土の上で葉が枯れていた。私が何を送れるかは分かっていた。記憶だった。記憶は水ではなかった。

記憶を送っても救えないものがあった。

それをその日知った。頭で知ったのではなく、鉢の前で知った。枯れた植物の記憶が三つあった。三つとも今日で、三つとも同じように終わった。

-----

研究室に行く回数が減った。

行けば覚えることが増えた。覚えることが増えれば確認することが増えた。部屋にいれば用が少なかった。用が少なければ今日が少なかった。

部屋は研究室より狭かった。ベッドと端末と窓一つ。窓の外に路地が見えたが、路地を見れば路地の記憶ができるので、カーテンを引いた。

部屋で端末を見た。書かれている今日と思い出す今日を合わせた。合わせるのに午前がいった。

研究室に行った。行った記憶があった。行かなかった記憶もあった。端末を見た。書かれていなかった。なら行かなかったのだ。手に机の感触があった。座った椅子の高さが体にあった。行かなかった研究室の椅子だった。

コーヒーを飲んだ。

飲まなかった。

味がした。

会議に行った。行くだろう。行ったか。

研究室の鍵を探した。

手に握っていた。

端末を見た。端末を見た記憶が幾つもあった。今しがた見たのがどれか選ばなければならなかった。選んだ。選んだのが合っているか端末を見た。

今朝何を食べたか考えた。幾つもの朝があった。全部今日だった。

あ。さ。に。コー。ヒー。を。

飲んだ。飲まなかった。

の。ん。

いつから数えていないのか思い出せなかった。

遅くない。そうだった気がする。

-----

夜だった。夜であることは窓の外で確認した。

保存をしたか思い出せなかった。保存した記憶はあった。その記憶が今日のものか分からなかった。端末を見た。時刻が刻まれていた。刻まれた時刻を読んだ。読んだものを忘れそうで、もう一度読んだ。

ルーターが机の上にあった。灯が点いていた。安定して。三年前に設計したそのまま。

壊れているのはルーターではなかった。

-----

## Ch.3

-----

朝だった。

一つだけの朝なのか確認するのに時間がかかった。端末に昨夜書いておいたものがあった。一行だった。読んだ。昨日が一つにまとまった。まとまった昨日をしばらく持って座っていた。

そして机へ行った。

ルーターに灯が点いていた。手が電源ボタンの上にあった。

切る前に保存しておくこともできた。そうすれば切った後も戻れた。ロードすればまた考えられた。もっといい方法を見つけられるかもしれなかった。ルーターは最後までその道を開けておいていた。それがルーターのすることだった。

そうしなかった。

ルーターを切った。それだけだった。

電源が落ちるのに三秒ほどかかった。三年かけて作ったものが切れるのには三秒で足りた。表示灯が死ぬと、それはまた外付けストレージのように見えた。机の上の物になった。

端末に計画書のファイルがあった。商用化日程、予想需要、試作仕様。三年間ルーターと一緒に育ったファイルだった。開かなかった。消しもしなかった。ただ置いておいた。

使ってみた人は私一人だった。それで十分だった。

-----

外に出た。

最後にいつ出たか記憶を探ろうとして、やめた。探る必要がないのが今日だった。

路地の端にパン屋があった。パンの匂いがした。実際の匂いだった。記憶ではないと確認する必要のない匂いだった。確認しなくてよいということを、確認してから知った。それも少しずつ減っていくはずだった。

信号が赤だった。待った。待つ間、何もしなかった。戻って直すことも、先に分かっている結末もなかった。青になった。渡った。

犬を連れた人が通り過ぎた。犬が私を一度見て通り過ぎた。そのことは一度起き、一度で終わった。

今日が一つだった。

それだけ確認した。パン屋の前で、信号の前で、路地で。今日が一つだということは確認するたびに事実だった。明日も一度だけ来るはずだった。一度来た明日は一度去るはずだった。

家に戻る道でカップを一つ買った。落とせば割れるカップだった。

よかった。

部屋に入ってカップを机の内側に置いた。角から遠い場所だった。それくらいでよかった。

夜になった。端末を開いた。今日あったことを一行書いた。書いてから、保存ボタンがあった画面を少し見た。

今日は保存しないことにした。

-----

*記憶ルーター — 終わり*
